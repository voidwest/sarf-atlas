# Sarf Atlas

Sarf Atlas is a prototype research workspace for Arabic morphology
representation studies. It is intended to organize research-facing material for
paper-series work while using Ember as the tooling and instrumentation layer.

This directory is not replacing Ember. Ember remains the backend, tooling,
artifact, pipeline, and instrumentation layer for extraction, probing, reports,
and reproducibility assets. Sarf Atlas is a thin workspace for paper-specific
research configs, notes, manifests, and experiment plans.

The current split is:

- `ember`: engineering, tooling, instrumentation, pipelines, and artifacts.
- `sarf-atlas`: Arabic morphology research workspace, paper configs, notes,
  experiment manifests, and paper-series planning.
- `gguf-parity-tools`: validation and parity harness.

This directory is experimental and local to the Ember repository for now. If the
shape proves useful, Sarf Atlas may later become a separate repository.
