# Examples Index

## Baseline Runner

Path: `examples/baseline_runner/`

Shows a generated `char_ngram` baseline config, split metadata, result
artifact, and summary artifact.

## Paper-Style Tiny Dataset

Path: `examples/paper_style/`

Contains the bundled morphology JSONL and experiment TOML used by docs and
tests.

## Paper 1-Style Reproduction Example

Path: `examples/paper1_reproduction/`

Runs the full Sarf workflow on tiny data with mock backend artifacts. This is
the main professor-packet demonstration path and does not claim full Paper 1
reproduction.

## Historical v0.1 Script

Path: `examples/run_v0_1_example.sh`

Kept as a historical smoke script. New users should start with
`sarf example-workflow` or the Paper 1-style example.
